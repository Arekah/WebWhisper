# WebWhisper 🎤
### Whisper to the Web. It Listens.

WebWhisper is a voice-first AI browser assistant built for visually impaired users. 
Simply speak a command and WebWhisper automatically navigates the web for you — 
completely hands free.

## 🌟 Features
- 🎤 Voice controlled browser automation
- 🔊 Reads page results aloud using text to speech
- 🧠 Powered by Amazon Nova 2 Lite on AWS Bedrock
- 💬 Remembers conversation context
- ⌨️ Text input alternative for accessibility
- 🌐 Supports Google, YouTube, Wikipedia, Amazon, Reddit and more

## 🛠️ Tech Stack
- Amazon Nova 2 Lite (AWS Bedrock)
- Python + Flask
- Playwright (browser automation)
- SpeechRecognition + pyttsx3

## 🚀 How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your AWS credentials to `.env`
4. Run: `python app.py`
5. Open `http://localhost:5000`
